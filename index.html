第一步：解锁bootloader

  不同机型解锁教程不一。

  1. 共同步骤“设置——关于手机——版本信息——点击7次版本号”。
  2. 返回“设置——系统——开发者选项”。
  3. 开发者选项里，设置OEM解锁打开，USB调试打开。
  4. 然后用各机型解锁工具解锁。

  ！！！解锁BL会清除手机数据！！！

第二步：提取boot&修补boot

  1. 下载系统当前版本的全量包 zip。
  2. zip里有boot.img直接用即可，没有看下一步。
  3. 下载payload-boot[^下载1]解包工具，把zip解压出来的payload.bin文件进行解包，得到boot。
  4. Android下载magisk[^下载2]——点击“安装”——“选择并修补一个文件”。

第三步：刷写boot

  1. pc 配置好adb[^Platform-Tools]环境，USB线连接。
  2. 手机进入fastboot模式：.
    • 方法一：在关机的情况下，同时按下power键和音量键。
    • 方法二：手机开启来后，手机插上USB线CMD输入： adb reboot bootloader
  3. 输入：fastboot flash boot
    magisk_patched-xxxxxx.img
  4. success——重启
